# Features

## Box Generator

- inside and outside dimension modes
- open-top boxes
- closed boxes
- finger-joint panel generation
- kerf and fit controls
- SVG export
- fit-test coupon export

## Lid Generator

- lift-off lid top plate
- four inner lip strips
- overhang control
- lip height control
- clearance control
- real-time SVG and 3D updates

## Divider Generator

- front-back divider count
- left-right divider count
- divider panels generated as laser-cut parts
- optional half-depth interlock slots
- divider parts included in SVG layout and 3D preview

## Panel Feature Editor

Shapes:

- round hole
- slot
- rectangle

Operations:

- cut
- engrave
- mark

Workflow:

- choose shape before placement
- enter size before placement
- click panel in SVG layout
- edit shape, operation, position, and size after placement
- delete selected feature

## 3D Preview

- Three.js assembly preview
- drag to rotate
- scroll to zoom
- exploded view slider
- optional 3D labels
- real-time add-on and feature updates

## SVG Export

The SVG export includes:

- generated panel outlines
- lid and divider parts
- cut, engrave, and mark feature geometry
- labels when enabled
- embedded metadata with normalized design settings
