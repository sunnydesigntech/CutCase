# CutCase Wiki

CutCase is a browser-based laser-cut finger-joint box designer with SVG export, rotatable 3D preview, lift-off lids, divider grids, and editable panel features.

## Current Status

CutCase currently supports:

- open-top and closed finger-joint boxes
- lift-off lids with top plate and inner lip strips
- front-back and left-right divider panels
- interlocking divider slots
- round holes, slots, and rectangular panel features
- cut, engrave, and mark operations
- SVG export with embedded metadata
- Three.js 3D preview and exploded inspection
- GitHub Pages deployment
