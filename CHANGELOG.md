## 0.1.0

- Added the CutCase static browser app.
- Added finger-joint box generation.
- Added open-top and closed-box panel generation.
- Added lift-off lid generator with top plate and inner lip strips.
- Added divider generator with interlocking slots.
- Added editable panel features: round holes, slots, and rectangles.
- Added cut, engrave, and mark feature operations.
- Added real-time SVG cut layout.
- Added rotatable Three.js 3D preview.
- Added SVG export with embedded metadata.
- Added fit-test coupon export.
- Added GitHub Pages deployment.
